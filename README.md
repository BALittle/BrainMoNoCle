# BrainMoNoCle
Brain Morphology Normative modelling platform for abnormality and Centile estimation: Brain MoNoCle

The mesial temporal lobe epilepsy (mTLE) data from the IDEAS dataset (Taylor et al. 2024) is in the IDEAS folder. This can be run with our normative model pipeline Brain MoNoCle (https://cnnplab.shinyapps.io/normativemodelshiny/).
